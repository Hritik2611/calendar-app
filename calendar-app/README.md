# Calendar App

A modern interactive calendar built using React and Tailwind CSS.

## Features

* 📆 Monthly calendar view
* 🎯 Date range selection (start & end)
* 🖌 Highlight selected dates
* 📝 Notes section with localStorage
* 📱 Responsive design

## 🛠 Tech Stack

* React.js
* Tailwind CSS
* JavaScript (ES6)

## 📦 Installation

```bash
npm install
npm run dev
```
